# 🌿 Carbon Footprint API Dokümantasyonu
> **v1.0** — Mobil Entegrasyon Kılavuzu | IzTek

---

## Genel Bilgi

| | |
|---|---|
| **Base URL** | `https://<domain>/api/v1` |
| **Auth** | `Authorization: Bearer <token>` |
| **API Version** | `/v1/` path prefix veya `api-version=1.0` |
| **Response Wrapper** | `Result<T>` \| `List<T>` \| `PagedResult<T>` |

### HTTP Durum Kodları

| Kod | Açıklama |
|-----|----------|
| `200` | Başarılı |
| `400` | Validasyon hatası |
| `401` | Token eksik veya geçersiz |
| `403` | Yetersiz yetki |
| `404` | Kayıt bulunamadı |

## 1. Users — `/api/v1/users`

> 🔐 **Yetki:** Authorize (Tüm kullanıcılar) + Admin bazı endpoint'ler

### Endpoints

| Metod | Endpoint | Parametreler | Açıklama |
|-------|----------|-------------|----------|
| `POST` | `/login` | Body: `LoginCommand` | Kullanıcı girişi. JWT token döner. *(AllowAnonymous)* |
| `POST` | `/password/forgot` | Body: `ForgotPasswordCommand` | Şifre sıfırlama SMS kodu gönderir. *(AllowAnonymous)* |
| `POST` | `/password/reset` | Body: `ResetPasswordCommand` | Yeni şifre belirler. SMS kodu doğrular. |
| `GET` | `/profile` | Token'dan UserId alınır | Kullanıcı profilini getirir. Rate limit uygulanır. |
| `GET` | `/detailed` | Query: `PageNumber, PageSize, SearchTerm, ShowDeleted` | Tüm kullanıcıların detaylı listesi. *(Admin)* |
| `POST` | `/donate-trees` | — | Birikmiş puanları ağaç bağışına dönüştürür. |
| `GET` | `/donation-history` | Query: `UserId` | Kullanıcının tüm bağış geçmişini listeler. |
